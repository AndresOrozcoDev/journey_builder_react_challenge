## Requirements

- [Node JS](https://nodejs.org/es).
- [Server backend]() running.

## Run local

Clone repository
```bash
    https://github.com/AndresOrozcoDev/journey_builder_react_challenge.git
    cd journey_builder_react_challenge
```

Install dependencies
```bash
    npm i
```

Run server
```bash
    npm run dev
```
